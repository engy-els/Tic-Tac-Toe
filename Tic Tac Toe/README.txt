Description: This folder contains 10 files, this one, that is README.txt, CellValue.java, GameState.java, TicTacToe.java, StudentInfo.java, TicTacToeGame.java, Utils.java, HumanPlayer.java, ComputerRandomPlayer.java, and Player.java

This assignment is a Tic Tac Toe game, that interacts with the user and allows two players (a player and a dumb computer) to play the traditional tic tac toe game. The board on which the game is played can be any size as well as the size to win, a player must get that number of consecutive boxes, in a row, column, or diagonal to win.
